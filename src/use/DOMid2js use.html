Pour utiliser DOMid2js il faut insérer la ligne:
<script src = "[path]/... /DOMid2js.js " > </script >
dans le html concerné par la conversion. Dans ces exemples, il suffit de faire un 'click' sur le body (dégradé ou bandeau).
La conversion est alors récupérable dans le clipboard (première balise style plus body).

Par défaut:
Le code du body est régroupé par id mais n'est pas remis en forme à l'exception les lignes vides qui sont supprimées.
Le premier style intégré (s'il existe) est codé et les littéraux sont encadrés de double quotes.

Il existe deux façons de passer les options à DOMid2js.
- Par le title de la balise script
<script src = "[path]/... /DOMid2js.js " title = "di2j template nostyle> </script >
- Ou en paramètre par l'URL d'appel du fichier html
[pathHtml].../myHtml.html?di2J=di2j template nostyle
